mpf2k cli

the cli program runs all scripts needed to manage library

-

USAGE

mpf2k <command> --flag

-

GENERATE

mpf2k generate - read all files in library root and generate metadata.toml and cover.image files based on config.toml and flags

--always-write-to-tracks = list of uppercase tags to always write to [[tracks]] section even if they are common for entire album
  e.g. --always-write-to-tracks=TRACKNUMBER,ARTIST,TITLE
